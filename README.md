# leisu-page-journal

一个用于归档与发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面文件，并利用 GitHub Pages 功能进行静态发布。这些页面内容各自独立，不依赖于特定域名或网站，适合作为临时页面、实验性页面、存档页面或展示性页面的托管场所。

## 目录说明

```
leisu-page-journal/
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面所需的通用资源文件（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，每个页面以单独文件或子目录形式存在。
- `assets/`：存放页面可能引用的公共资源，如样式表、脚本、字体或图片。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文件，可直接在浏览器中打开或通过 GitHub Pages 访问。
- 页面内容不指向任何具体网站或服务，仅作为静态内容展示或信息存档。
- 归档的页面可能包含但不限于：个人笔记、工具界面、示例页面、临时页面等。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请遵循现有目录结构。
- 提交内容前请确保 HTML 文件格式正确，且不包含敏感信息或恶意代码。
- 若需删除或替换某个页面，请直接在 `pages/` 目录下操作并提交变更。
- 欢迎通过 Issue 或 Pull Request 提出改进建议或报告问题。

## 许可

本项目未指定具体开源许可证，内容仅供个人归档与展示使用。如需引用或分发，请提前联系维护者。
